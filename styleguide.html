Universal Robots Styleguide

Vi er 6 studerende fra Multimediedesign på UCL, der arbejder på et projekt vedrørende Universal Robots' brugerflade, hvorfor vi har udviklet denne styleguide. Denne living styleguide er en videre guide til designet på vores løsning til UR i Projekt 4, og indeholder derfor både design-elementer samt code snippets.

Logo

Anvendelse: Logo skal bruges på samtlige sider til venstre i navigationsbaren.

Code Snippet (HTML):

                
<div class="logo">
    <a href="#intro">
        <img src="./img/ur-logo.png" alt="logo">
    </a>
</div>

Code Snippet (CSS):

                
.logo {
    display: flex;
    flex: 2;
    width: 40%;
    margin-left: 7%;
    margin-right: 3%;
  }

.logo img {
    width: 200px;
    padding-bottom: 10px;
  }

Typografi

Anvendelse: Heading 1 bruges til de største og vigtigste overskrifter, Heading 2 bliver brugt til underoverskrifterne, og Brødteksten er det resternede tekst på siderne.

Code Snippet (CSS):

                
html {
  font-family: Roboto, sans-serif, arial;
}

h1 {
    font-size:47px;
}

h2 {
    font-size:30px;
}

p {
    font-size:16px;
}

Farver

HEX #FFFFFF

HEX #303030

HEX #56A0D3

Anvendelse: De tre ovenstående farver bruges igennem hele websitedesignet, og afspejler de farver Universal Robots bruger på deres hjemmeside. Den hvide farve (HEX #fff) bruges til baggrund samt tekst elementer som har brug for en anden kontrastfarve, eksempelvis teksten på de blå knapper. Den mørkegrå farve (HEX #303030) bruges til al tekst på hvid baggrund samt tekst på billedelementer. Den bruges lige så til footer og designelementer som skal skabe moduler og opdeling i designet. Den blå farve (HEX #56a0d3) bruges til knapper, links, citater og ikoner samt i designelementer i eksempelvis videoafspilleren på sitet.
